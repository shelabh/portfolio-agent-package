# Legacy Code Areas

The supported public contract for this repository is:

- `PortfolioAgent`
- local FAISS indexing
- `create_app(...)`

Archived or deferred areas have been isolated under explicit legacy namespaces such as:

- `legacy_rag`
- `legacy_vectorstores`
- `legacy_llm_backends`
- `legacy_persistence`
- `legacy_finetune`
- `legacy_finetuning`
- `legacy_security`
- `legacy_checkpoint`

They are not part of the supported public API.
