# Recruiter Example

This example is intentionally not part of the supported public surface today.

The repository is currently standardized around one canonical SDK path:
- `PortfolioAgent`
- local ingestion into `FAISSVectorStore`
- querying through `PortfolioAgent.query(...)`

If you want a recruiting-specific workflow, build it on top of the documented SDK rather than the legacy example scripts. Start from the root [README](../../../README.md).
