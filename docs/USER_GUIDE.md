# User Guide

The supported user experience for this repository is intentionally narrow:

1. Create a `PortfolioAgent`
2. Ingest content
3. Query it
4. Optionally expose it through the minimal FastAPI app

For the exact code paths, use:
- [README](../README.md)
- [Quickstart](QUICKSTART.md)
- [Manual E2E Verification](MANUAL_E2E.md)

Older guides that referenced `PortfolioAgent` methods or HTTP endpoints that do not exist anymore have been removed from the supported path.
