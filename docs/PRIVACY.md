# Privacy Policy

## Overview

The Portfolio Agent is designed with privacy-first principles, implementing comprehensive data protection measures to ensure user privacy and regulatory compliance.

## Data Collection

### What We Collect

The Portfolio Agent may collect and process the following types of data:

#### Personal Information
- **Contact Information**: Email addresses, phone numbers (when provided)
- **Professional Information**: Resume data, work experience, skills
- **Usage Data**: Interaction logs, query history, preferences

#### Technical Information
- **System Data**: Log files, error reports, performance metrics
- **Configuration Data**: User preferences, system settings
- **Metadata**: File information, timestamps, source attribution

### What We Don't Collect

We explicitly do not collect:
- Financial information (credit cards, bank accounts)
- Government-issued IDs (SSN, passport numbers)
- Biometric data
- Health information
- Location data (unless explicitly provided)

## User Rights

### GDPR Rights

Under the General Data Protection Regulation (GDPR), users have the following rights:

#### Right to Access
- Request copies of personal data
- Information about processing purposes
- Details about data recipients

#### Right to Rectification
- Correct inaccurate personal data
- Complete incomplete information
- Update outdated information

#### Right to Erasure ("Right to be Forgotten")
- Delete personal data
- Stop processing
- Remove from all systems

#### Right to Restrict Processing
- Limit data processing
- Suspend certain operations
- Maintain data for legal purposes

#### Right to Data Portability
- Export personal data
- Transfer to another service
- Machine-readable format

#### Right to Object
- Object to processing
- Opt-out of marketing
- Withdraw consent
